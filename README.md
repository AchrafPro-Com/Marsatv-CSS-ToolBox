# MarsaTvToolbox

A modern, interactive collection of CSS utility generators built for developers. Craft beautiful styles in seconds with real-time previews and one-click copy.

🔗 **Live Demo:** [marsatv.com](https://marsatv.com)

---

## ✨ Features

MarsaTvToolbox provides **7 powerful CSS generators** in a sleek dark-themed interface:

| Tool | Description |
|------|-------------|
| **Box Shadow** | Generate layered drop shadows with full color, opacity, and spread control. Supports both solid color and image URL previews. |
| **Gradient** | Create linear and radial gradients with 8 direction options and dual-color blending. |
| **Border Radius** | Fine-tune each corner independently — from sharp edges to smooth organic shapes. |
| **Text Shadow** | Add depth to typography with customizable blur, offset, and color shadows. |
| **Glassmorphism** | Design frosted glass UI elements with blur, saturation, opacity, and border controls. |
| **Transform** | Apply 2D transformations — rotate, scale, skew, and translate — with real-time preview. |
| **Filter** | Apply visual effects including blur, brightness, contrast, grayscale, hue-rotate, saturate, and sepia. |

## 🎨 Design Highlights

- **Dark Mode UI** — Easy on the eyes with a deep slate palette
- **Glassmorphism Cards** — Frosted glass panels with backdrop blur
- **Animated Mesh Background** — Subtle floating gradient orbs
- **Real-time Preview** — Instant visual feedback as you adjust sliders
- **One-click Copy** — Copy generated CSS to clipboard with a single click
- **Fully Responsive** — Works seamlessly on desktop, tablet, and mobile
- **SPA Navigation** — Smooth transitions between the home grid and individual tools

## 🚀 Getting Started

No build step required. Simply open the `index.html` file in any modern browser:

```bash
git clone https://github.com/Marsatv-CSS-ToolBox/MarsaTvToolbox.git
cd MarsaTvToolbox
# Open index.html in your browser
```

Or serve it locally:

```bash
npx serve .
# or
python -m http.server 8080
```
